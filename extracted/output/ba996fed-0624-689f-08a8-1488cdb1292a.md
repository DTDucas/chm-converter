# Remarks
If [LockProportions](2eda8038-9240-00f5-ffa3-a65d8b4df5f2.md "LockProportions Property") is true, then changes to the Height will also result in changes to the [Width](c135b323-bdcf-4945-6f7b-f2da08bb41fe.md "Width Property").
