# Remarks

This is available only for constraints of type [ToOtherRebar](b9417455-214c-30ef-e055-ad17ff1e0db3.md).

Valid [TargetRebarConstraintType](b5eb20bf-e887-41b4-f15c-0c1860799a8b.md) are: 

  * [RebarPlane](b5eb20bf-e887-41b4-f15c-0c1860799a8b.md)
  * [Edge](b5eb20bf-e887-41b4-f15c-0c1860799a8b.md)
  * [OutOfPlaneExtent](b5eb20bf-e887-41b4-f15c-0c1860799a8b.md)
  * [StartOfBar](b5eb20bf-e887-41b4-f15c-0c1860799a8b.md), only if the RebarShape has a definition of [RebarShapeMultiplanarDefinition](47a3135c-ce53-c041-f551-0795767eaa41.md) with [!:RebarShapeMultiplanarDefinition::IsStartConnectorPressent]
  * [EndOfBar](b5eb20bf-e887-41b4-f15c-0c1860799a8b.md), only if the RebarShape has a definition of [RebarShapeMultiplanarDefinition](47a3135c-ce53-c041-f551-0795767eaa41.md) with [!:RebarShapeMultiplanarDefinition::IsEndConnectorPressent]
