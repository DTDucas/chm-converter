#### Parameters

    Type: [Autodesk.Revit.DB..::..FamilyInstance](0d2231f8-91e6-794f-92ae-16aad8014b27.md) The column to attach. If the target is a beam or brace, the column will be checked to see if it is slanted. Otherwise, this argument is not used and may be omitted.

    Type: [Autodesk.Revit.DB..::..Element](eb16114f-69ea-f4de-0d0d-f7388b105a16.md) A proposed target element for a column attachment. 

# Remarks

Valid targets are roofs, floors, and ceilings. Beams and braces are also valid targets, except for slanted columns.
