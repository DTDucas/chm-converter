# Remarks

A modifier consists of a type (from [ModifierType](14da29ca-e466-9c3f-7a5b-5988a0e0ef6b.md) and an optional separator. Apply a modifier to one or more [ExportLayerInfo](88a99694-968a-99f7-870a-f46737bd5927.md) objects to modify the layer name that will be assigned when a Revit object is exported.
