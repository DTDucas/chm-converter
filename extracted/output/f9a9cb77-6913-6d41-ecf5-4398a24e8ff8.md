# Remarks

The UniqueId can be used to store an identifier in an external database and to retrieve the same element in the future if it still exists. This id can be passed to the Document's Element property to retrieve the element. The UniqueId is stable across upgrades and workset operations such as Save To Central, while the [ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) property may change.
