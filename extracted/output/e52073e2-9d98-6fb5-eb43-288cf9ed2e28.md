# Remarks

IndependentTag represents single-category tags, multi-category tags, material tags, and zone tags. IndependentTag is also the base class for other annotations like keynote tags and span symbols. Room, area and space tags are not derived from IndependentTag. [SpatialElementTag](0a20cdd6-6e44-bc77-a4c3-2d35470ba911.md) for more information.
