    Type: [Autodesk.Revit.DB..::..ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) This ViewFamilyType will be used by all elevations hosted on the new ElevationMarker.

    Type: System..::..Int32 This view scale will be automatically applied to new elevations created on the ElevationMarker. The scale is the ratio of true model size to paper size.
