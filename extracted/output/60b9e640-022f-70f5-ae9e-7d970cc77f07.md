# Remarks

When getting this property, it returns a copy of the [ExportLayerInfo](88a99694-968a-99f7-870a-f46737bd5927.md) from the table. In order to make changes to the [ExportLayerInfo](88a99694-968a-99f7-870a-f46737bd5927.md) and use those settings during export, set the modified [ExportLayerInfo](88a99694-968a-99f7-870a-f46737bd5927.md) back into the table using the same key.
