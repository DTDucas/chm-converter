# Remarks

The value for this property must be positive and cannot be Infinity. This value is interpreted in device-independent units (1/96th inch) measurement [device-independent units](ms-help://MS.VSCC.v90/MS.MSDNQTR.v90.en/fxref_presentationframework/html/40f9e1d0-afab-e8eb-5eb5-f0499be66eca.md). The default value is 200.
