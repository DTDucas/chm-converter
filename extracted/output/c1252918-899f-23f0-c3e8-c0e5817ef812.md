# Remarks

For hosted load only LoadOrientTo.Project and LoadOrientTo.HostLocalCoordinateSystem are permitted. For non-hosted load only LoadOrientTo.Project and LoadOrientTo.WorkPlane are permitted. To determine if load is hosted use [IsHosted](76965c6d-473a-9ad9-8a72-baa7a47b055a.md) property.
