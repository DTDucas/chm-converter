# Remarks

Only DisplayStyle::Wireframe and DisplayStyle::HLR values are supported. Default forced value is DisplayStyle::HLR unless the view has DisplayStyle:Wireframe. 

See notes for 2D export for views in non-Wireframe display style in [!:Autodesk::Revit::DB::IExportContextBase] and [IExportContext2D](a4578846-6ecf-e354-668d-96d8ef5d1a32.md).
