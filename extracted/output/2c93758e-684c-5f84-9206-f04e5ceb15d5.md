Returns ids of elements which belong to that sketch. 

# Remarks

Following elements can belong to a sketch: [ModelCurve](f15a85c2-3aee-9055-f9f8-9001b47fcefb.md), [ReferencePlane](e7003ec7-1dbe-50a2-fb3d-a83a5a3b5b9f.md), [Dimension](210f88be-e3c5-26a4-7dd8-3296f6725cce.md). To get matching between [Curve](400cc9b6-9ff7-de85-6fd8-c20002209d25.md) from [Profile](1fdfbff7-dfea-5d08-3583-89d0218eff3c.md) and [ModelCurve](f15a85c2-3aee-9055-f9f8-9001b47fcefb.md), use [!:Autodesk::Revit::DB::Curve::Reference::ElementId].
