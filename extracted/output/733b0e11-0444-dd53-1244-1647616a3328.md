# Remarks

The referenced element may be any model element, including FamilyInstance or FormElement. The reference must be of type ElementReferenceType.REFERENCE_TYPE_SURFACE, and the surface must be of type [Plane](6a6ee978-f114-558d-3c69-00d289aa855f.md). The Reference's UVPoint property is ignored.
