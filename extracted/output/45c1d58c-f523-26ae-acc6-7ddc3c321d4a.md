# Remarks

Please checkout the workset before executing this method. The method may fail in some situations that mentioned in [CanDeleteWorkset(Document, WorksetId, DeleteWorksetSettings)](6a120bcb-6b51-f8c4-2f59-e21b15c31b6a.md). Another failure case is the Transaction failure due to "Deleting all open views in a project is not allowed."
