#### Parameters

document
    Type: [Autodesk.Revit.DB..::..Document](db03274b-a107-aa32-9034-f3e0df4bb1ec.md) The document. 

multistoryStairsId
    Type: [Autodesk.Revit.DB..::..ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) The id of the [MultistoryStairs](8b07cbff-013c-889f-8807-703e63a91923.md) that contains the stairs which will be the host of this railing. The stairs should have no associated railings yet. 

levelId
    Type: [Autodesk.Revit.DB..::..ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) The level of stairs on which the new railing will be created. The level should be one of levels defining the [MultistoryStairs](8b07cbff-013c-889f-8807-703e63a91923.md) element. 

railingTypeId
    Type: [Autodesk.Revit.DB..::..ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) The railing type of the new railing to be created. 

placePosition
    Type: [Autodesk.Revit.DB.Architecture..::..RailingPlacementPosition](893811f2-d3ba-8ef4-a397-322416cfcbfd.md) The placement position of the new railing. 

#### Return Value

The new railing instances successfully created on the stairs.
