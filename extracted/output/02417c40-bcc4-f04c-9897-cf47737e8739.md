#### Parameters

materialId
    Type: [Autodesk.Revit.DB..::..ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) The material id returned from [GetMaterialIds(Boolean)](6011352e-151b-b8ac-14cc-45970f2fe5ad.md). 

usePaintMaterial
    Type: System..::..Boolean If true, this material id was returned as a paint material from [GetMaterialIds(Boolean)](6011352e-151b-b8ac-14cc-45970f2fe5ad.md) and the area returned should be calculated from paint applied to the element. If false, this material id was returned as a non-paint element material from [GetMaterialIds(Boolean)](6011352e-151b-b8ac-14cc-45970f2fe5ad.md) and the area is calculated from the element geometry and layers. 

#### Return Value

The area of the material for this element. Returns 0.0 if the material id is not a part of this element.
