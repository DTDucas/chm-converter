Identifies if radial or diameter dimensions will be shown for the bends between segments. Radial or diameter dimesions for arc segments will be shown by default. This property has a menaning only if [BendDiameterDimensionsEnabled](5e06f0e4-2980-403f-79d8-a58c70dd5a3e.md "BendDiameterDimensionsEnabled Property") is set to true.
