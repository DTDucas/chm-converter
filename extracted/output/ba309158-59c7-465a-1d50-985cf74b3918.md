Modifies the input Solid preserving only the volume on the positive side of the given Plane. The positive side of the plane is the side to which Plane.Normal points.

#### Parameters

solid
    Type: [Autodesk.Revit.DB..::..Solid](7a3b5ac1-c66d-9f81-a11d-9bcd4e026295.md) The input Solid to be cut. This object cannot be obtained directly from a Revit element. This means that [IsElementGeometry](be3ad18d-a9d3-25ed-6200-4f71d3cd4754.md) cannot be true. 

plane
    Type: [Autodesk.Revit.DB..::..Plane](6a6ee978-f114-558d-3c69-00d289aa855f.md) The cut plane. The space on the positive side of the normal of the plane will be intersected with the input Solid. 

# Remarks

This operation modifies the original input Geometry objects.
