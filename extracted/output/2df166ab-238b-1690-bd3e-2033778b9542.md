# Remarks

If %reference% points to linked document: 

  * The [GetReference()()()()](62e0779b-25c1-b83a-0a13-ed2bf9cececc.md) method of the newly created subelement will return the local reference in the linked document.
  * The [Document](5606267f-30e4-bd6c-8e06-43ad1f495585.md) property of the newly created subelement will return the linked document.
