#### Parameters

reference
    Type: [Autodesk.Revit.DB..::..Reference](d28155ae-817b-1f31-9c3f-c9c6a28acc0d.md)A candidate reference in selection operation.

position
    Type: [Autodesk.Revit.DB..::..XYZ](c2fd995c-95c0-58fb-f5de-f3246cbc5600.md)The 3D position of the mouse on the candidate reference.

#### Return Value

Return true to allow the user to select this candidate reference. Return false to prevent selection of this candidate.

# Remarks

If an exception is thrown from this method, the element will not be permitted to be selected.
