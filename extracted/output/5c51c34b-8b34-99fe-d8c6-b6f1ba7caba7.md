# Remarks

The element is a [SpotDimension](f3c633ac-1595-cb8d-5c1b-66eb3eefb433.md). The element's category is [OST_AlignmentStationLabels](ba1c5b30-242f-5fdc-8ea9-ec3b61e6e722.md) or, if in a set of labels, [OST_AlignmentStationLabelSets](ba1c5b30-242f-5fdc-8ea9-ec3b61e6e722.md). The element's type is a [SpotDimensionType](06ffc197-308a-a350-6dd7-6f812e175bb6.md) with [DimensionStyleType](130b0264-615d-610e-38e0-4ce2a8e2aecd.md) equal to [AlignmentStationLabel](130b0264-615d-610e-38e0-4ce2a8e2aecd.md). The element's [Origin](df8b9dc6-9d36-ac2b-04cf-816d88f039b8.md) is a point on the tessellated representation of an alignment. To get the precise point on the alignment's curve, use [GetPointAtStation(Double)](1b4cc73b-dc00-0439-5480-fd7979b1e106.md) with input obtained from [Station](1558579e-cdee-03ca-58b1-5630fe0fa0c1.md).
