#### Parameters

shellLayerType
    Type: [Autodesk.Revit.DB..::..ShellLayerType](75c640b9-9904-7882-43fc-a4dfc25ff53c.md) If ShellLayerType.Exterior return the number of exterior shell layers (or top shell layers for a roof, floor, or ceiling type). If ShellLayerType.Interior return the number of interior shell layers (or bottom shell layers for a roof, floor, or ceiling type). 

#### Return Value

The number of shell layers in the interior or exterior shell, as specified by shellLayerType. 

# Remarks

There will always be at least one core layer, i.e. one layer which is not a shell layer. You can change the shell/core layer boundary using [SetNumberOfShellLayers(ShellLayerType, Int32)](8b8ea4e3-e697-6176-92c0-dc2687723a71.md).
