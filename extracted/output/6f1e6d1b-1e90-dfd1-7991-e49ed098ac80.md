#### Parameters

document
    Type: [Autodesk.Revit.DB..::..Document](db03274b-a107-aa32-9034-f3e0df4bb1ec.md) The document. 

straightId
    Type: [Autodesk.Revit.DB..::..ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) The element identifier of the straight to be cut in to. 

partId
    Type: [Autodesk.Revit.DB..::..ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.md) The element identifier of the part to align and cut in with. 

position
    Type: [Autodesk.Revit.DB..::..XYZ](c2fd995c-95c0-58fb-f5de-f3246cbc5600.md) The position to move the parts insertion point to. 

rotation
    Type: System..::..Double The rotation in radians. 

slope
    Type: System..::..Double The slope value to flex to match if possible in fractional units (eg.1/50). Positive values are up, negative are down. Slopes can only be applied to fittings, whilst straights will inherit the slope from the piece it is connecting to. 

flip
    Type: System..::..Boolean Flip the part to allow for flow direction. 

#### Return Value

True if the alignment succeeds, false otherwise.
