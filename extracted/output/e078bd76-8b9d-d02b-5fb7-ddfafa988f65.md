# Remarks

The resulting collection may contain both "individual" labels (created via [Create(Alignment, View, AlignmentStationLabelOptions)](eb69d2d4-5a55-6402-ae7b-d1049fdba2d4.md)) and "labels in set" (created via [CreateSet(Alignment, View, AlignmentStationLabelSetOptions)](bbb3fb20-cbc6-f6aa-cc23-ae7ad73747b3.md)). These labels have categories respectively [OST_Alignments](ba1c5b30-242f-5fdc-8ea9-ec3b61e6e722.md) and [OST_AlignmentStationLabelSets](ba1c5b30-242f-5fdc-8ea9-ec3b61e6e722.md).
