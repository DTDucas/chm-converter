#### Parameters

panel
    Type: [Autodesk.Revit.DB..::..Element](eb16114f-69ea-f4de-0d0d-f7388b105a16.md) The panel to be changed, it can be a type of [Panel](ad561307-a19c-9a8a-728d-5646e90b451b.md) or [Wall](b5891733-c602-12df-beab-da414b58d608.md). 

newSymbol
    Type: [Autodesk.Revit.DB..::..ElementType](ffb18296-0448-559c-580c-7857cbcdc094.md) The new symbol, it may be of [PanelType](3a8ad72e-5aa7-8fef-10ba-72041fe47346.md) or [WallType](aa685433-b426-5e4f-bee1-e3487bb59518.md) when the panel is hosted in a curtain wall. The new symbol can only be of type [PanelType](3a8ad72e-5aa7-8fef-10ba-72041fe47346.md) if the Panel is hosted in a curtain system. 

#### Return Value

If operation succeeds, the modified panel element is returned.
