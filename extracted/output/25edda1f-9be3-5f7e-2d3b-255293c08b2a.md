# Remarks

If there is no existing MultiReferenceAnnotationType a new one will be created, otherwise an exception will be thrown. Use [Duplicate(String)](b0e7d5d5-f33a-8ff2-b471-78a213f06ef5.md) to create all further MultiReferenceAnnotationTypes. The default MultiReferenceAnnotationType will use the document's current default linear dimension style and will have Structural Rebar as its reference category. No tag type will be set for the new MultiReferenceAnnotationType.
