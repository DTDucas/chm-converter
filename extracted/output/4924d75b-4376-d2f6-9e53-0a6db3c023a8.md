# Remarks

Multiple dimensions may need to be set in order to reach the desired outcome. The document must be regenerated before the fabrication part can be used. Check [ValidationStatus](9d208301-607f-45b0-d5af-e20b2e6a92ed.md) after regeneration to see if the part is valid for fabrication.
