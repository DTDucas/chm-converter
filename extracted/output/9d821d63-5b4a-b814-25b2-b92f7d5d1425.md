# Remarks

The stable representation can be used to preserve and restore the reference later in the same Revit session or even in a different session where the same document is present. Use [ParseFromStableRepresentation(Document, String)](dc168535-2688-83da-429f-a2d018ff4b43.md) to restore the reference. The representation is based on the internal Revit structure and is not intended to be parsed expect by [ParseFromStableRepresentation(Document, String)](dc168535-2688-83da-429f-a2d018ff4b43.md).
