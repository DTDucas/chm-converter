| Member name | Description |
| --- | --- |

| None | None of the characters in the range has the specified format. |

| Mixed | The characters in the format have mixed formatting. |

# Remarks
The character formatting includes 
  * Bold
  * Italic
  * Underline
  * SuperScript
  * Subscript
  * All Caps
